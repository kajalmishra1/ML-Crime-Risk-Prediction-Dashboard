# 🚔 Intelligent Crime Analysis using Machine Learning

### Predict • Analyze • Prevent Crime using Data-Driven Insights

---

## 📌 Project Overview

This project is a **Machine Learning-based crime risk prediction system** that classifies regions into **Low, Medium, or High risk categories**.

It uses historical crime data along with factors like police strength and arrest records to support **data-driven decision making** in crime analysis and prevention.

The goal is to demonstrate how ML can be applied to **real-world problems** by extracting meaningful insights from structured datasets.

---

## ⚙️ Key Features

* 🔍 Crime risk classification (**Low / Medium / High**)
* 📊 Crime count estimation
* 💡 Explanation of predictions
* 🚨 Preventive recommendations

---

## 🧠 Machine Learning Model

* **Algorithm Used:** Random Forest Classifier
* Handles structured data effectively
* Reduces overfitting and improves accuracy

## 🖥️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/kajalmishra1/ML-Crime-Prediction-Dashboard.git
cd ML-Crime-Prediction-Dashboard
```

### 2. Install dependencies

```bash
pip install pandas numpy scikit-learn
```

### 3. Run the application

```bash
python app.py
```

**OR (GUI version)**

```bash
python gui.py
```
